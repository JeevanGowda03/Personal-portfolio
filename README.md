# Jeevan H S — Personal Portfolio

Personal portfolio website of **Jeevan H S**, an aspiring software engineer and CSE student building privacy-focused, secure applications across Android, cloud, and AI-driven systems.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite 5** for build tooling
- **Tailwind CSS** + **shadcn/ui** for styling
- **React Router** for routing

## Project Structure

```
src/
├── components/
│   ├── portfolio/    # Hero, About, Skills, Projects, Experience, Contact, Footer, Navbar
│   └── ui/           # Reusable UI primitives
├── pages/            # Route pages
├── lib/              # Helpers (smooth scroll, utils)
└── hooks/            # Custom React hooks
```
